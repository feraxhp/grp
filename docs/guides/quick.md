
## Installation

### Windows

In windows you can use `winget`
~~~bash
winget install --id feraxhp.grp
~~~

### Fedora

~~~bash
sudo dnf install <see releases for the url>
~~~

> It will be available trough copr in the future.

### Ubuntu
go to [releases](https://github.com/repos/feraxhp/grp/releases/latest) page for the latest deb package.
```bash
wget -O girep.deb <URL_DEL_PAQUETE>
sudo apt install ./girep.deb
```

### Other and Macos

There is not a implemented `CI/DI` So you need to [build it from source](/guides/build).
